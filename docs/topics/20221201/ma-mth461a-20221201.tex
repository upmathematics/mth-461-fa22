\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\item
  In a communication system each data packet consists of \(1000\) bits. Due to the noise, each bit may be received in error with probability \(0.1\). It is assumed bit errors occur independently. Let us define \(X_i\) as the indicator random variable for the \(i\)th bit in the packet. That is, \(X_i=1\) if the \(i\)th bit is received in error, and \(X_i=0\) otherwise. Then the \(X_i\)'s are i.i.d. and \(X_i \sim Bernoulli(p=0.1)\). Find the probability that there are more than \(120\) errors in a certain data packet.

  Recall that for a Bernoulli random variabl \(E[X] = p\) and \(Var(X)=p(1-p)\).
\item
  Suppose that in a population, the wait times at bus stops can be represented as an Exponential random variable with parameter \(\lambda = 0.5\). This can be written as \(X \sim Exponential(0.5)\).

  \begin{enumerate}
  \def\labelenumii{\alph{enumii}.}
  \tightlist
  \item
    What is the population mean? Recall that \(E[X] = \frac{1}{\lambda}\) for an exponental random variable \(X\).
  \item
    Modify the \texttt{clt.R} script to simulate the exponential random variable, create a histogram, and draw a line that represent the population mean.
  \item
    Simulate \(N\) random samples where each sample have \(k\) items from your simulations in (b), and compute the means of these samples. Plot your simulations. As you increase \(N\) while \(k\) is fixed, what happens to the mean of means? Explain in terms of the Central Limit Theorem.
  \end{enumerate}
\end{enumerate}
